# **App Name**: AI Product Portfolio

## Core Features:

- Project Showcase: Display AI product builds with detailed descriptions and results.
- Skills Summary: Highlight expertise in AI and product management.
- AI-Powered Portfolio Review Tool: Offer instant AI feedback to anyone reviewing your portfolio. It can be instructed to assess technical achievements or managerial skills.
- Contact Form: Allow visitors to easily send messages.

## Style Guidelines:

- Primary color: Deep Indigo (#4B0082), evokes intelligence, future-thinking, and sophistication.
- Background color: Very light lavender (#F0F8FF) that subtly complements the primary and accent color without causing distraction.
- Accent color: A vibrant violet (#A020F0) to draw attention to important UI elements like links and calls to action.
- Headline font: 'Space Grotesk' sans-serif. Body font: 'Inter' sans-serif.
- Simple, modern icons to represent AI concepts and skills.
- Clean and structured layout, balancing text and visual elements for an intuitive flow.
- Subtle transitions for content sections.
